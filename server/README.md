# PuAc TW Project

Simple REST API with pure nodejs and mongoDB as NOSQL DB
## Run project 
**Method 1 **
1. Install Nodejs
2. Open Folder in VSCode
3. Open terminal
4. Run **npm i**
5. Run **npm start**
6. The server will be live at **http://localhost:5000**
**http://localhost:5000/items** - you should see some data
